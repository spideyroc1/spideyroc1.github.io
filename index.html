What is Dragonet?

Dragonet is a Minecraft server software that supports both Minecraft: PC Edition, and Minecraft: Pocket Edition. It currently supports: A vast majority of Bukkit plugins, multi-world, world generation, placing/destroying blocks. It is currently maintained by @TheMCPEGamer, and the Dragonet Team.

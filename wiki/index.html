Installation Instructions

In order to use Dragonet, you must have Java installed on your system. Then, download a pre-compiled binary from the Dragonet website or compile a JAR from the source code found on our GitHub repository. Once you have the JAR file, create a new folder ("directory") named something similar to 'Dragonet' and move your JAR file there. Once you have done that, open the folder you just created and open a new command/terminal window. In your terminal window, input the command 'java -Xmx{0}MB -jar {1}' (where {0} is the max amount of RAM you wish to allocate for the server and {1} is the name of the JAR file.) Dragonet should then start, and create all the folders and the configs automatically.

Building from Source

To build Dragonet from source, you will need two things: Gradle, and a copy of Dragonet's source code, which can be found at our GitHub repository. Once you have both of those things, unzip both of the archives and navigate to the root directory of the source code, and open a new terminal window there. Once you have done that, drag the 'gradle.bat' file from the bin file of the Gradle archive into the terminal window and press Enter. It should then start to compile the Dragonet JAR file. Once it finishes, navigate to the 'build\libs\' folder and the file with 'all' in it's name is the compiled JAR file that you can use.
